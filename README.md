# A股历史K线数据集

本项目使用 akshare 库获取 A 股所有股票最近 10 年的日K线数据，并保存为 CSV 格式。

## 项目结构

```
quantBox/
├── src/
│   └── download_kline.py    # 数据下载脚本
├── data/
│   └── *.csv                # K线数据文件
└── README.md                # 说明文档
```

## 环境要求

- Python 3.9+
- akshare
- pandas
- tqdm
- requests

## 安装依赖

```bash
pip install akshare pandas tqdm requests
```

## 使用方法

### 1. 网络要求

**重要**: akshare 的数据源为国内网站（东方财富网），需要在中国大陆网络环境下运行。

如果无法访问：
- 设置代理：`set HTTP_PROXY=http://127.0.0.1:7890`
- 或使用 VPN

### 2. 运行脚本

```bash
python src/download_kline.py
```

### 3. 数据说明

- **数据来源**: 东方财富网 (via akshare)
- **时间范围**: 2015年1月1日至今
- **股票范围**: 所有A股（已过滤ST股票和退市股票）
- **存储格式**: CSV文件
- **文件命名**: `{股票代码}_{股票名称}.csv`

### 4. CSV 文件字段

| 字段名 | 说明 |
|--------|------|
| 日期 | 交易日期 |
| 开盘 | 开盘价 |
| 收盘 | 收盘价 |
| 最高 | 最高价 |
| 最低 | 最低价 |
| 成交量 | 成交量 |
| 成交额 | 成交额 |
| 振幅 | 振幅(%) |
| 涨跌幅 | 涨跌幅(%) |
| 涨跌额 | 涨跌额 |
| 换手率 | 换手率(%) |
| 股票代码 | 股票代码 |
| 股票名称 | 股票名称 |

## 注意事项

1. **下载时间**: 全部A股约5000只股票，全部下载可能需要数小时
2. **网络稳定**: 确保网络连接稳定，脚本会自动重试失败请求
3. **数据量**: 10年数据每个股票约2500条记录，5000只股票总计约1250万条记录
4. **磁盘空间**: 预计需要约 2-5 GB 存储空间

## 常见问题

### Q: 下载失败怎么办？
A: 检查网络连接，确保可以访问国内网站。可设置代理后重试。

### Q: 可以只下载部分股票吗？
A: 可以修改 `get_stock_list()` 函数，添加过滤条件。

### Q: 数据是实时更新的吗？
A: 不是，这是历史数据。获取最新数据需要每个交易日运行脚本。

### Q: 为什么有些股票下载失败？
A: 可能原因：股票已退市、数据源不稳定、请求过快被限流。脚本会自动跳过失败的股票。
